DOCUMENT TECHNIQUE — MINI PROJET SEO (IRAN NEWS)
=================================================

1) INFORMATIONS GÉNÉRALES
-------------------------
- Nom du projet : Iran News (FrontOffice + BackOffice)
- Repository : SEO-Practise
- Technologies : PHP 8.2, MySQL 8, Apache, Docker Compose
- URL FrontOffice (FO) : http://localhost:8080/front/
- URL BackOffice (BO) : http://localhost:8080/admin/login.php
- Num ETU : [A REMPLIR ICI]


2) OBJECTIF DU PROJET
---------------------
Ce projet permet de publier et consulter des articles sur l’actualité du conflit Iran / Moyen-Orient.
Le FrontOffice présente les articles aux visiteurs (SEO, URL propres, accessibilité),
le BackOffice permet l’administration des contenus (CRUD articles + gestion images).


3) MODÉLISATION DE LA BASE DE DONNÉES
-------------------------------------
Base : iran_news

Tables principales :
- dim_temps
  - id_temps (PK), date, annee, mois
- dim_categorie
  - id_categorie (PK), nom, slug
- dim_auteur
  - id_auteur (PK), nom, prenom
- fait_article
  - id_article (PK), id_temps (FK), id_categorie (FK), id_auteur (FK)
  - titre, contenu, slug, nb_vues, is_active, date_publication
- fait_article_image
  - id_image (PK), id_article (FK), image_url, alt_text, is_main
- admin
  - id_admin (PK), username, password, created_at

Relations :
- 1 catégorie -> N articles
- 1 auteur -> N articles
- 1 date (dim_temps) -> N articles
- 1 article -> N images

Remarque :
- Le champ contenu dans fait_article stocke du HTML (balises h2/h3/p/ul/li).
- image_url est stocké sous forme de chemin relatif front (ex: /front/assets/img/article-01.jpg).

7) PROCÉDURE DE LANCEMENT (RAPIDE)
----------------------------------
1) Démarrer les services :
   docker-compose up -d --build

2) Réinitialiser complètement la base (si nécessaire) :
   docker-compose down -v
   docker-compose up -d --build

3) Accéder aux interfaces :
   - FO : http://localhost:8080/front/
   - BO : http://localhost:8080/admin/login.php


8) POINTS SEO / ACCESSIBILITÉ / PERFORMANCE (RÉSUMÉ)
-----------------------------------------------------
- SEO : title et meta dynamiques, URLs réécrites, alt des images, structure de titres.
- Accessibilité : focus visible, contraste amélioré, hiérarchie de titres corrigée.
- Performance : cache statique, gzip, preload image LCP, optimisations CSS.
